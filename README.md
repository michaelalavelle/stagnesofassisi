# St. Agnes of Assisi Fraternity (OFS)

Static website for the St. Agnes of Assisi Fraternity of the Secular Franciscan Order (Tau Cross Region).

## Commands (Windows PowerShell)

```powershell
npm install
```

```powershell
npm run dev
```

```powershell
npm run build
```

## Notes

- Content for announcements and events lives in `src/content/` as Markdown.
- Pages are in `src/pages/` and share layout in `src/layouts/BaseLayout.astro`.
